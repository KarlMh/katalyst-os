================== EdScribe – Katalyst Editor ==================

Commands (& prefix):

&s Save: write buffer to file

&q Quit: exit editor; warns if unsaved changes exist

&u Undo: revert last edit (multi-line supported)

&r Redo: reapply undone edits

&v Void: clear entire buffer

&c 3 -> 6 Copy lines 3–6 to fragment buffer

&p Paste: insert fragment buffer at cursor

&f pattern Find: search text; highlights all matches

&g pattern Goto pattern: jump cursor to matching line

&d 5 -> 8 Delete lines 5–8

&peek folder Peek: view folder contents without leaving editor

&pulse Execute: run current line as Katalyst command

&nn Toggle line numbers

&stats Show buffer info: lines, chars, RAM, fragment, undo depth

&map Mini-map: ASCII overview of buffer

&theme color Change editor theme/colors

&sparkle Fun typing animation/effects

&preview Live preview of scripts or formatted content

Editor Features:

- Multi-level undo/redo stack

- Fragment buffer visualization

- Line highlighting (current/search/recent edits)

- Auto-indentation & bracket matching

- Auto-completion for Katalyst commands

- Mini-map & scroll preview

- Jump markers & foldable sections

- Peek-as-you-type & live execution (&pulse)

- Split-view file linking

- Custom themes & ASCII/typing effects

- Status bar: filename, cursor line, RAM, fragment buffer, undo stack

- Interactive help overlay

- Optional preview pane for scripts/markdown

Navigation:

- Arrow keys: move cursor

- gg / G: jump to start/end of buffer
